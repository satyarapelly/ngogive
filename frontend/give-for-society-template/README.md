# Give for Society NGO Starter Template

This is an original NGO web app starter template inspired by professional nonprofit information architecture (hero, trust signals, programs, impact stats, stories, donate CTA), built so you can quickly customize branding and data.

## Structure
- `index.html`: Single-page starter layout
- `assets/css/styles.css`: Base design system and responsive layout
- `assets/js/app.js`: Lightweight impact counter animation
- `content/`: Website content blocks
- `proposals/`: Proposal and concept note starters

## Run locally
Open `index.html` directly in a browser or serve the folder with any static server.
